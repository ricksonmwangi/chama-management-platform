# Genje Group

## Production Milestone

This repository contains the complete Genje Group platform, consisting of both the backend API and the frontend application.

Genje Group is a modern Chama ledger built to virtualise all chama activities and transactions.

## Repository Structure

```text
backend/   → Backend API
frontend/  → Frontend Application
```
