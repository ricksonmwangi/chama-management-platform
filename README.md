# Chama Management Platform

A full-stack Chama Management Platform that enables savings groups to manage members, contributions, meetings, loans, repayments, and M-Pesa transactions securely.

## Current Features

- JWT Authentication
- Role-Based Authorization
- Member Management
- Contributions Management
- Meetings Management
- Loan Processing
- Loan Eligibility Checks
- Loan Approval & Rejection
- Loan Repayments
- Outstanding Loan Balance Calculation
- Dashboard Statistics
- Audit Logging
- M-Pesa Daraja Integration

## Technology Stack

### Backend

- Node.js
- Express.js
- MySQL
- JSON Web Tokens (JWT)
