# Genje Group Frontend

## Overview

The Genje Group Frontend is the user-facing application for the Genje Group Chama ledger. It provides an intuitive and responsive interface for managing our chama group activitits and transactions.

The application communicates with the backend REST API and delivers a secure, modern experience for administrators and members.

## Features

* User Authentication
* Role-Based Access Control
* Dashboard Analytics
* Member Management
* Contributions Management
* Loan Management
* Loan Repayments
* Meeting Management
* Settings Management
* Audit Log Viewing
* M-Pesa Integration
* Responsive User Interface

## Technology Stack

* React
* React Router
* Axios
* Modern JavaScript (ES6+)
* CSS / UI Components

## Status
